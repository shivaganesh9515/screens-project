## 1. Project Overview

A full-stack cloud digital signage platform for managing networks of physical display screens (buses, autos, kiosks, menu boards). Organizations upload media, build playlists and zone-based templates, schedule content to screens or screen groups, and monitor playback through a live dashboard and analytics.

This quotation covers the **complete platform**, including:

- The core signage engine (already built)
- The franchise/advertiser expansion (in progress)

---

## 2. Scope of Work

### 2.1 Core Platform (Completed)

- User authentication (login, signup, password reset)
- Organization and role management (admin / editor / viewer)
- Screen management — registration, pairing, screen groups, heartbeat/online status
- Media library — image/video upload, folders, tags
- Playlist builder — drag-and-drop ordering, per-item duration
- Zone templates — split-screen layout editor, playlist-per-zone binding, drag-to-resize
- Scheduling — assign playlists/templates to screens or groups, time windows, recurrence
- Player application — pairing flow, content playback, play logging
- Analytics dashboard — playback reporting, CSV export
- Settings — organization profile, team invites, logo, billing placeholder

### 2.2 Franchise/Advertiser Expansion (In Progress)

- Three-tier role structure: **Main Admin → Franchise → Advertiser**
- Franchise and advertiser account management
- Ad creation and two-tier approval workflow
- Screen metadata upgrades and GPS location tracking
- Live map view of screen locations and status
- Role-based dashboards for each tier (Main Admin / Franchise / Advertiser)
- Extended analytics for ad performance and screen status history
